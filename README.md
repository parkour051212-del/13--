 반복문, 조건문, 함수, 리스트 활용 등을 익히는 데 도움이 되도록 작성된 예제 모음
각 문제는 독립적인 기능을 수행하며, 입력값을 받아 결과를 출력하는 형태로 구성되어 있다

포함된 함수 목록
#39 홀수 출력 함수 — print_odds(n)
0부터 n까지의 정수 중 홀수만 출력
for 반복문으로 0에서 n까지 순회
i % 2 == 1 조건으로 홀수 판별

#40 3의 배수 판별 함수 — print_multiple_of_three(num)
입력한 정수가 3의 배수라면 그대로 출력
num % 3 == 0 조건을 활용한 단순 판별

#41 네 개의 숫자 최댓값·최솟값 찾기 — find_max_min(a, b, c, d)
네 개의 입력값 중 최댓값과 최솟값을 반환
리스트로 묶어 max(), min() 내장 함수를 사용
반환값은 (최댓값, 최솟값) 형태의 튜플

#42 n까지의 홀수 연속 출력 — print_odds_upto(n)
0부터 n까지의 모든 홀수를 공백으로 이어 출력

#43 팩토리얼 계산 — factorial(n)
입력된 n(0 이상의 정수)의 팩토리얼 값을 계산해 반환
1부터 n까지 곱셈 반복
결과 반환

#44 곱이 30 이상인 값들의 합 — sum_products_over_30(i, j)
두 수 i, j가 주어졌을 때
1 ≤ x ≤ i, 1 ≤ y ≤ j 범위의 모든 (x*y) 중
곱이 30 이상인 값들만 합산
이중 for문
x * y >= 30 조건을 만족할 때만 누적

#45 누적합 출력 — cumulative_sum(lst)
리스트의 요소를 앞에서부터 더한 누적합을 순서대로 출력
